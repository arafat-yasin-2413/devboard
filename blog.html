Question-1: What are the different ways to select an element in the DOM?

We can select html elements using DOM in 5 different ways. They are :
a) getElementById()
b) getElementsByTagName()
c) getElementsByClassName()
d) querySelector()
e) querySelectorAll()

Question-3: What is event delegation in the DOM?

Event delegation হচ্ছে এমন একটি পদ্ধতি যার মাধ্যমে কোনো HTML এলিমেন্টের প্যারেন্টের উপর addEventListener বসিয়ে তার child গুলোকে অ্যাক্সেস কর যায়। এক্ষেত্রে প্রতিটা child এ event listener বসানোর প্রয়োজন হয় না। আমরা প্যারেন্টে event listener বসিয়ে callback ফাংশনে event কে পাস করি। তারপর event.target দিয়ে child গুলোকে ধরি। এটা event bubble এর মাধ্যমে কাজ করে।

Question-4: What is event bubbling in the DOM?

Event Bubbling হলো DOM এর এমন বৈশিষ্ট্য যেখানে কোনো child এলিমেন্টের উপর event ট্রিগার হয়ে সেই event টি তার প্যারেন্ট এলিমেন্টের দিকে ধাপে ধাপে যেতে থাকে। event টি উপরের দিকে document না পাওয়া পর্যন্ত যেতে থাকে। এটা অনেকটা পানির নিচ থেকে উপরের দিকে bubble উঠে যাওয়ার মতো করে কাজ করে। এভাবে কোনো একটা event নিচ থেকে উপরের দিকে bubble করতে করতে যায়।

Question-5: How do you create, add, and remove elements using JavaScript?

create:

const newP = document.createElement('p');
newP.innerText = "A quick brown fox had a nice tail.";


add:

const parentDiv = document.getElementById('parent-div');
const h3 = document.createElement('h3');
h3.innerText = "DOM is the life to my website";
parentDiv.appendChild(h3);

remove:

const items = document.getElementById('items');
const child = document.getElementById('child-3');
items.removeChild(child);
